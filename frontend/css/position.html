position property:

used to control the position of html elements

        1.position:static-this is default position of all html elements(we cannot use top,left, bottom & right properties with the position)
        2.position:relative-this will position the elements from original position
        3.position:absolute-this will positionthe elements from the nearest position ancestor
        4.position:fixed-it will fix the html elements
        5.position:sticky-it will stick the html elements when we scroll the pages
